# 图书馆管理系统 - Netlify 部署指南

本文件夹包含图书馆管理系统的演示版本，可以直接上传到 Netlify 进行部署。

## 部署步骤

1. **登录或注册 Netlify 账号**
   - 访问 [Netlify 官网](https://www.netlify.com/)
   - 点击右上角的 "Sign Up" 或 "Log In"

2. **创建新站点**
   - 登录后，点击 "Add new site" 按钮
   - 选择 "Drop manually" 选项

3. **上传文件**
   - 将整个 `library_system_demo` 文件夹拖拽到上传区域
   - 或者点击 "Browse" 按钮选择整个文件夹

4. **等待部署完成**
   - Netlify 会自动构建和部署你的站点
   - 部署完成后，你会获得一个随机生成的 Netlify 域名

5. **自定义域名（可选）**
   - 如果你有自己的域名，可以在 Netlify 控制台中设置自定义域名

## 注意事项

- 这是一个**静态演示版本**，所有数据都存储在浏览器的 localStorage 中
- 数据仅在本地保存，刷新页面或关闭浏览器后数据会保留，但不会在不同设备间共享
- 如果需要完整的动态功能，请参考 `DEPLOYMENT_GUIDE.md` 中的完整动态系统部署方案

## 如何使用

部署完成后，你可以：
1. 使用学生账号 `123456` 或 `654321` 登录系统
2. 浏览书库、借阅图书、查看借阅记录
3. 使用管理员账号 `admin`（密码 `admin123`）登录管理页面
