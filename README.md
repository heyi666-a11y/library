# 广东北江中学图书馆管理系统 - 前端

## 项目简介
这是一个现代化的图书馆管理系统前端，同时服务于学生自助借还书与图书馆老师后台管理两大场景。系统采用了专业、优雅的数据驱动设计，提供了清晰的用户界面和流畅的操作体验。

## 核心功能

### 学生终端界面
- **身份验证**：支持学号/密码登录
- **借书功能**：扫描图书条形码完成借阅，提供实时反馈
- **还书功能**：直接扫描条形码归还图书
- **我的借阅**：查看当前借阅、历史记录和逾期提醒

### 管理后台
- **数据仪表盘**：实时展示在馆图书数、今日借阅量、活跃借阅数、逾期未还数等关键指标
- **可视化图表**：近30天借阅趋势图、热门书籍TOP10、图书分类分布
- **图书管理**：强大的搜索和筛选功能，支持添加、编辑、批量导入图书
- **读者管理**：按年级、班级查看学生信息和借阅档案
- **借阅记录**：完整的操作日志，支持多维度筛选和导出
- **系统设置**：配置借阅参数、发布公告等

## 技术栈
- **HTML5**：语义化标签，响应式设计
- **CSS3**：Flexbox、Grid布局，渐变色彩，动画效果
- **JavaScript (ES6+)**：模块化设计，DOM操作
- **Chart.js**：数据可视化图表
- **Font Awesome**：图标库

## 设计特点

### 视觉设计
- 现代化的渐变色彩方案（紫色/蓝色为主色调）
- 清晰的信息层级，良好的对比度
- 圆角设计和微妙阴影，提升界面质感
- 统一的组件样式和交互效果

### 用户体验
- 直观的操作流程，减少学习成本
- 明确的状态反馈（加载中、成功、失败）
- 平滑的过渡动画
- 响应式设计，适配不同屏幕尺寸

## 快速开始

1. 克隆或下载项目文件到本地
2. 使用HTTP服务器启动项目（例如Python的http.server）：
   ```bash
   cd library_system
   python -m http.server 8000
   ```
3. 在浏览器中访问：http://localhost:8000
